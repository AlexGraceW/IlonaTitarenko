## GitHub Pages (статический сайт)

### Структура
- `/index.html` — главная
- `/works/index.html` — работы
- `/contacts/index.html` — контакты
- `/assets/data/*.json` — контент
- `/assets/js/*.js` — логика подстановки контента

### Деплой на GitHub Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` / Folder: `/ (root)`

После `git push` сайт обновится автоматически.
